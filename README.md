# Student Result Management System

This is a simple Python project to calculate student results using CSV data.

## Features
- Calculates total marks
- Calculates percentage
- Assigns grades

## Technologies Used
- Python
- Pandas

## How to Run
1. Clone the repository
2. Install dependencies:
   pip install pandas
3. Run:
   python student_result.py
